## 💻 Projeto
Projeto desenvolvido na **Trilha Ignite** do **NLW eSports**.

Esse projeto é uma aplicação web e mobile, com servidor. Essa é uma aplicação relacionada a eSports (jogos online, no ambiente competitivo).
